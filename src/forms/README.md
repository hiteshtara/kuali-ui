# Forms

Form component with a "grid" layout to use in Kuali's UI.

## `<Form />`

### PropTypes
| name      |type       | default     | required      |
|-----------|-----------|-------------|---------------|
| showGrid  | boolean   | false       | no            |

### Example Usage
```js
import { Form } from 'kuali-ui'

export function SomeComponent() {
  return (
    <Form>
        <div>Some really nice form goes here!</div>
    </Form>
  )
}

```
